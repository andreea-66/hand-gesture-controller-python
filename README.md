# Real-Time Hand Gesture Media Controller

This project uses OpenCV and MediaPipe to recognize hand gestures in real-time and control a media player.

## Features
- Gesture recognition (Start, Stop, Pause, OK, Peace, Rock)
- Visual effects and sound feedback
- Dynamic volume control using finger distance

## Technologies
- Python
- OpenCV
- MediaPipe
- NumPy

## How to run
```bash
pip install -r requirements.txt
python main.py
