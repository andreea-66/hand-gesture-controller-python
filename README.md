# Real-Time Hand Gesture Media Controller

A real-time hand gesture recognition system built with **OpenCV** and **MediaPipe** that allows users to control a virtual media player using hand gestures. The application uses computer vision techniques to detect hand landmarks, interpret gestures, and provide visual and audio feedback in real time.

---

## Features

- Real-time hand tracking using MediaPipe  
- Gesture-based media control:
  - STOP  
  - PLAY (START)  
  - PAUSE  
  - OK  
  - PEACE (with animated GIF)  
  - ROCK (special visual effects)  
- Volume control using the distance between fingers  
- Visual feedback:
  - Progress bar  
  - Volume bar  
  - Gesture detection UI  
- Audio feedback using system beeps  
- Dynamic visual effects (lightning, overlays, animations)  

---

## Technologies Used

- Python  
- OpenCV  
- MediaPipe  
- NumPy
